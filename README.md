# Go-Web
Modern Web Application with Golang

# Stack
- HTTP Middleware [Echo](https://echo.labstack.com/)
- Language [Go](https://golang.org/) +1.8
- ORM library [gorm](https://github.com/jinzhu/gorm)
- Configuration [env](github.com/caarlos0/env)

# Getting Started

```
$ go get ./...
$ docker-compose up
$ go run main.go
```
